1) Does ANSI recognize the data type DATE?

--> ANSI SQL does recognize the data type DATE. It's a standard data type used to store dates without any time information. It's supported by many RDBMS that adhere to ANSI SQL standards, such as MySQL, PostgreSQL, Oracle, SQL Server, and others.
